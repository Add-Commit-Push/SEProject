# Desktop Application
***
This is the resturant management dash.

### Version
0.1.0

## Usage

### Installation

```sh
$ npm install
```

### Serve
To run

```sh
$ npm start
```

### Package & Build

For Windows

```sh
$ npm run package-win
```

For Mac

```sh
$ npm run package-mac
```

For Linux

```sh
$ npm run package-linux
```
